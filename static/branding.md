# LockBay Branding Guidelines

## Logo
- **Primary Logo**: `static/lockbay-logo.png`
- **Format**: PNG with transparent background support
- **Usage**: All marketing materials, email headers, documentation

## Color Palette
Based on the official LockBay logo:

### Primary Colors
- **Deep Blue/Teal**: `#1e3a8a` (Primary brand color)
- **Turquoise**: `#06b6d4` (Accent color from wave design)
- **Dark Navy**: `#1e293b` (Text and borders)

### Secondary Colors
- **Light Gray**: `#f8fafc` (Backgrounds)
- **Success Green**: `#10b981` (Positive actions)
- **Warning Amber**: `#f59e0b` (Cautions)
- **Alert Red**: `#ef4444` (Errors/Critical)

## Typography
- **Primary Font**: System fonts (-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto)
- **Brand Name**: Always "LockBay" (one word, capital L and B)
- **Tagline**: "Safe trading on Telegram!"

## Visual Elements
- **Security Focus**: Emphasize padlock, shield, and protection imagery
- **Wave Elements**: Subtle wave patterns from logo design
- **Clean Design**: Minimal, professional, trustworthy appearance

## Usage Guidelines
- Logo should maintain clear space equal to the height of the "L" in LockBay
- Minimum size: 100px width for digital use
- Never distort or alter the logo proportions
- Always use on contrasting backgrounds for readability
